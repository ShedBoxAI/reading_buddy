# Privacy Policy for ReadingBuddy

**Last updated:** December 27, 2025

## Overview

ReadingBuddy is a browser extension that helps users understand text on webpages by providing AI-powered explanations. We are committed to protecting your privacy.

## Data Collection

### What we collect

ReadingBuddy collects and processes the following data **only when you explicitly request an explanation**:

- **Highlighted text**: The text you select on a webpage
- **Page context**: Surrounding text, page title, and URL to provide better explanations

### What we store locally

The following data is stored locally in your browser using Chrome's storage API:

- Your selected AI provider (Ollama, OpenAI, or Anthropic)
- API keys you enter (stored locally, never transmitted to us)
- Model preferences
- Extension enabled/disabled state

### What we do NOT collect

- Browsing history
- Personal information
- Analytics or usage data
- Any data when you're not actively using the extension

## Data Transmission

### Local AI (Ollama)

When using Ollama, **all processing happens on your machine**. No data is sent to external servers. This is the most private option.

### Cloud AI (OpenAI / Anthropic)

When using OpenAI or Anthropic:

- Highlighted text and page context are sent to the respective API
- Data is transmitted securely via HTTPS
- We do not store or have access to this data
- Please review their privacy policies:
  - [OpenAI Privacy Policy](https://openai.com/privacy)
  - [Anthropic Privacy Policy](https://www.anthropic.com/privacy)

## Data Sharing

We do **NOT**:

- Sell your data to third parties
- Share your data with third parties (except the AI provider you select)
- Use your data for advertising
- Track your browsing activity

## Data Storage

- All settings are stored locally in your browser
- API keys are stored locally and never transmitted to us
- No data is stored on our servers (we don't have servers)

## Your Rights

You can:

- Disable the extension at any time
- Clear all stored data by removing the extension
- Use Ollama for completely local, private processing
- Review our open-source code on GitHub

## Open Source

ReadingBuddy is open source. You can review exactly what data is collected and how it's used:

https://github.com/ShedBoxAI/reading_buddy

## Changes to This Policy

We may update this privacy policy from time to time. Changes will be posted to this page with an updated date.

## Contact

If you have questions about this privacy policy, please open an issue on GitHub:

https://github.com/ShedBoxAI/reading_buddy/issues

**Summary**: ReadingBuddy only processes text you explicitly highlight. Use Ollama for complete privacy. We don't collect, store, or sell your data.
